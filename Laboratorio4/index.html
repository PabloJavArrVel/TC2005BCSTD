¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Diferencias:

Propósito: Java es un lenguaje de programación de propósito general que se utiliza principalmente para el desarrollo de aplicaciones empresariales, mientras que JavaScript se utiliza principalmente para el desarrollo de aplicaciones web y móviles.

Tipo de lenguaje: Java es un lenguaje de programación compilado, lo que significa que el código fuente se compila en un archivo ejecutable antes de que se pueda ejecutar, mientras que JavaScript es un lenguaje de programación interpretado, lo que significa que el código fuente se ejecuta directamente en el navegador o en el entorno de ejecución correspondiente.

Sintaxis: Java y JavaScript tienen una sintaxis similar debido a su herencia de C, pero JavaScript es más flexible y menos estricto en cuanto a la sintaxis y la semántica, lo que facilita la escritura de código.

Uso de objetos: En Java, todo es un objeto y se rige por un sistema de tipos fuertes y estáticos, mientras que en JavaScript, todo es un objeto y se rige por un sistema de tipos dinámicos y débiles.

Similitudes:

1- Sintaxis: Java y JavaScript tienen una sintaxis similar debido a su herencia de C.
2- Ambos lenguajes son orientados a objetos y permiten la creación de clases y objetos.
3- Ambos lenguajes tienen una amplia gama de bibliotecas y frameworks disponibles para simplificar el proceso de desarrollo de aplicaciones.
4- Ambos lenguajes son muy populares y ampliamente utilizados en la industria de la tecnología.


¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

getDate: devuelve el día del mes (entre 1 y 31) para la fecha especificada.
getMonth: devuelve el mes (entre 0 y 11) para la fecha especificada, donde 0 representa enero y 11 representa diciembre.
getFullYear: devuelve el año (con cuatro dígitos) para la fecha especificada.
getHours: devuelve la hora (entre 0 y 23) para la fecha especificada.
getMinutes: devuelve los minutos (entre 0 y 59) para la fecha especificada.

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

push: agrega uno o más elementos al final del arreglo y devuelve la nueva longitud del arreglo.
pop: elimina el último elemento del arreglo y lo devuelve.
slice: devuelve una copia de una parte del arreglo especificada por un rango de índices.
join: une todos los elementos del arreglo en una cadena y devuelve la cadena resultante.
indexOf: devuelve el índice de la primera ocurrencia de un elemento especificado en el arreglo, o -1 si no se encuentra el elemento.

¿Cómo se declara una variable con alcance local dentro de una función?

Para declarar una variable con alcance local dentro de una función en JavaScript, se utiliza la palabra clave var, let, o const seguida del nombre de la variable y su valor (opcional) dentro de las llaves de la función.
